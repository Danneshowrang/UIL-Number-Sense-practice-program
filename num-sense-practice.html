Instructions

How to Use:

1. Choose a practice mode from the main menu

2. Solve problems mentally - no calculator!

3. Type your answer in the input box

4. Click "Submit Answer" to check

5. Click "Next Question" to continue

6. Use 'Skip' if you don't know the answer

7. Use 'Quit' to end any session early

8. Review missed problems to learn from mistakes


Stopwatch Controls:

• ▶ Start: Begin timing

• ⏸ Pause: Pause timing

• ↺ Reset: Reset to 00:00:00

(Stopwatch automatically starts for untimed modes)


Keyboard Shortcuts:

• Enter: Submit answer

• N: Next question (after submission)

• Esc: Skip problem

• Q: Quit practice


For Estimation Problems (*):

Answers within 5% of exact answer are scored correct

All Problem Types

Tier 1 - Fundamental:

• Addition (3-4 digit numbers)

• Subtraction (with borrowing)

• Multiplication (2-digit × 2-digit)

• Division (even division)

• Fraction ↔ Decimal conversions

• Percent ↔ Fraction conversions

• Squares (11² to 25²)

• Order of operations

• Roman numerals

Tier 2 - Patterns & Tricks:

• Multiplication by 11 trick

• Remainders (divisibility rules)

• Averages (mean)

• Percent of a number

• Unit conversions (inches/feet/yards)

Tier 3 - Algebra & Geometry:

• Solve for x (linear equations)

• Binomial squares (patterns)

• Pythagorean triples

Tier 4 - Estimation (* problems):

• Square root approximation

• Large addition chains
